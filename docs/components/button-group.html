Components

Button Group

Button Group is a container for grouping buttons.

Example:

Group buttons are used to group a set of related actions together.


<div class="btn-group">
  <button class="btn primary">Button</button>
  <button class="btn primary">Button</button>
  <button class="btn primary">Button</button>
</div>

Mixed Group Button:



<div class="btn-group">
  <button class="btn primary">Button</button>
  <button class="btn secondary">Button</button>
  <button class="btn success">Button</button>
  <button class="btn danger">Button</button>
  <button class="btn warning">Button</button>
  <button class="btn info">Button</button>
</div>

Group Button (Outline style):



<div class="btn-group">
  <button class="btn outline-primary">Button</button>
  <button class="btn outline-secondary">Button</button>
  <button class="btn outline-success">Button</button>
  <button class="btn outline-danger">Button</button>
  <button class="btn outline-warning">Button</button>
  <button class="btn outline-info">Button</button>
</div>
