Components

Text Field

Text fields are used to collect information from the user. They are mainly used in forms.

Example:

Easy.css provides a text field component which are easy to implement. Just add which type of input field you want to use outline or Filled. Thats it.

 
<input type="text" class="outline" placeholder="Outline Text Field" />
<input type="text" class="filled" placeholder="Filled Text Field" />

Password:

Password fields are used to collect sensitive information from the user.They are mainly used in forms.This style can also be used in password fields.

 
<input type="password" class="outline" placeholder="Outline Password Field" />
<input type="password" class="filled" placeholder="Filled Password Field" />


Textarea:

Textarea are mainly used to collect long text from the user like user address or summery .

 
<textarea class="outline" cols="30" rows="10" placeholder="Outline Textarea"></textarea>
<textarea class="filled" cols="30" rows="10" placeholder="Filled Textarea"></textarea>
