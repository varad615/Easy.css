Loaders

Loaders are used to indicate that something is loading.

Example:

Loaders are used to indicate that something is loading. Loaders can be used to indicate that a page is loading, or that a user is waiting for a response.
Easy.css make it easy to build a loader. Just add the class loader to the div to show the loader.


<div class="loader"></div>

Loader variety :

Easy.css provides a variety of loaders. You can use the class loader- to add a variety of loaders.


<div class="loader primary"></div>
<div class="loader success"></div>
<div class="loader danger"></div>
<div class="loader warning"></div>
<div class="loader info"></div>
