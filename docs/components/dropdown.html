Components

Dropdown

Dropdown is a button that opens a menu.

Example:

Dropdown is a button that opens a list of selectable options that can perform an action. Dropdown options can also be links that can redirect user to other page.


<select class="select-outline">
  <option selected>Options</option>
  <option value="1">One</option>
  <option value="2">Two</option>
  <option value="3">Three</option>
</select>
<select class="select-filled">
  <option selected>Options</option>
  <option value="1">One</option>
  <option value="2">Two</option>
  <option value="3">Three</option>
</select>

Dropdown disabled :

Dropdown buttons can also be kept disabled. This is useful when you want to keep the dropdown open but don't want to allow the user to select an option.
Easy.css provides a disabled state for dropdown buttons with change in logo which tells the user that the dropbox is disabled.


<select class="select-outline" disabled>
  <option selected>Options</option>
  <option value="1">One</option>
  <option value="2">Two</option>
  <option value="3">Three</option>
</select>
<select class="select-filled" disabled>
  <option selected>Options</option>
  <option value="1">One</option>
  <option value="2">Two</option>
  <option value="3">Three</option>
</select>
